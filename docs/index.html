Projetos

1º Semestre • 01/2023 • Consulta-vid

Projeto desenvolvido por alunos do 1º semestre do curso de Desenvolvimento de Software Multiplataforma da Fatec São José dos Campos – Prof. Jessen Vidal. A aplicação web analisa dados da covid longa nas cidades do Vale do Paraíba, ajudando profissionais de saúde, pesquisadores e o público em geral a entenderem seu impacto no sistema de saúde local. O back-end usa Python com Flask e MySQL para gerenciar os dados. A plataforma oferece informações detalhadas sobre casos, hospitalizações, sintomas e tratamentos, focando na covid longa para apoiar estratégias eficazes de enfrentamento.
